Detailed Description

The ZoomSDK class is a class that exposes a Zoom API Rest Client.

Access to this class and all other components of the ZoomSDK can be granted by including <ZoomSDK/ZoomSDK.h> in your source code. This class provides a class method sharedSDK which provides a preconfigured SDK client
